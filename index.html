Space Shooter

This project is a space shooter game built in Unity using C#. The goal of the game is to stay alive as long as possible, avoiding and shooting space ships to stay alive. There are 3 different levels, the easy level has slow moving ships that disappear at the bottom of the screen. The medium level has the ships bouncing back and forth, only disappearing when being shot. The hard level has the enemy ships gaining speed as they bounce off the borders, along with asteroids that fly in any direction across the screen that hurt you. If too many enemys appear on the hard level you automatically lose. The skills I gained from this game was rotational display and physics, dynamic health display, and creative level design.

Boggle with AI Solver

This is a playable word-solving game made in Python using Pygame. It is similar to the game Boggle, but without the timer. You can try for as long as you want to find all of the words; however, if you press the "give up" button it will show you all of the possible words from the current board. The program uses a BFS to search all possible moves on the board to ensure completeness. It dynamically displays the words in alphabetical order and updates the score and word list. After it is done, it pauses for 5 seconds for the user to see the max score, then it resets so you can play again.

Karate Time - HTML5 Fighter Game

This was my first ever game project. I coded this using HTML5, CSS, and JavaScript. I created all of the art and animations, which is why the girl character's animation is buggy. I learned the importance of creating spritesheets with equivalent spacing and sizing in order to do proper calculations for the animation. I made the hitbox match the width of the characters, which was also buggy due to the characters' different sizes and using one universal method for creating the hitbox. If I were to go back and work on this again, I would redesign the characters to make sure they are the same size, as well as create a separate CSS file to handle all of the styling that is currently hard-coded into the HTML. While this isn't my best project, it is my first, and I am proud of how I have used the lessons from it to improve my other works.
