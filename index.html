Boggle with AI Solver

This is a playable word solving game made in Python using pygame. It is similar to the game Boggle but without the timer. You can try for as long as you want to find all of the words; but if you press the give up button it will show you all of the possible words from the current board. The program uses a BFS to search all the possible moves on the board to ensure completeness. It dynamically displays the words in alphabetical order and updates the score and word list. After it is done it pauses for 5 seconds for the user to see the max score, then it resets so you can play again.

Karate Time - HTML5 Fighter Game

This was my first ever game project, I coded this using HTML5, CSS, and JS. I created all of the art and animations which is why the girl characters animation is buggy. I learned the importance of creating spritesheets with equivilant spacing and sizing in order to do proper calculations for the animation. This lesson also translated into how hitboxes worked. I made the hitbox match the width of the characters, which also was buggy due to the charcters different sizing and one method for creating the hitbox. If I were to go back and work on this again, I would redesign the characters to make sure they are the same size, as well as create a seperate CSS file to handle all of the styling that is currently hard coded into HTML. While this isn't my best project, it is my first and I am proud of how I have used the lessons here to improve my other works.
